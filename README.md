# Molde
> Open modular hardware platform for rapid physical prototyping.

## Overview
Molde is an open-source ecosystem of interchangeable hardware modules that snap onto a powered base panel.
Buttons, displays, rotary encoders, sensors, LEDs and other modules can be rearranged in seconds without rewiring or soldering.
The goal is to make physical prototyping as flexible as software development.

## Why?
Traditional prototyping often requires rewiring every time the hardware changes.
Molde aims to simplify this process by providing reusable plug-and-play hardware modules that can be freely rearranged while sharing power and communication through a common backplane.
